## Installation

* Télécharger cmake https://cmake.org/ 
(l'executable doit être dans $PATH)
* Installer VisualStudio avec compilateur c++ https://visualstudio.microsoft.com/fr/
* Redémarrer la console
* `mkdir build`
* `cd build`
* `cmake ..`
* `cmake --build .`
* `./Debug/casses_briques-(cli|gui)[.exe]`
